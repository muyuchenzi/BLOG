## 迭代器

```python
#迭代器（Iterator） 是同时实现__iter__() 与 __next__() 方法的对象
#可迭代对象有很多，list tuple dict 但是却不一定是迭代器。
list_alpha = [i for i in range(100)]


class MyIterator:
    def __init__(self):
        self.num = 0

    def __iter__(self):
        return self

    def __next__(self):
        return_num = self.num
        self.num += 2
        if self.num == 100:
            raise StopIteration
        else:
            return return_num


itertor_alpha = MyIterator()
for _ in itertor_alpha:
    print(_)
```
